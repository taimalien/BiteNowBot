# @BiteNowBot Deploy Guide
# 100% FREE — no Anthropic key needed

## You need:
- Fresh Telegram bot token from @BotFather
- Free GitHub account (github.com)
- Free Railway account (railway.app)

---

## Step 1 — Put files on GitHub
1. Go to github.com → sign in → click "+" → New repository
2. Name it: bitenowbot
3. Click "Create repository"
4. Drag index.js and package.json into the page
5. Click "Commit changes"

## Step 2 — Deploy on Railway
1. Go to railway.app → "New Project" → "Deploy from GitHub repo"
2. Select bitenowbot
3. Go to Variables tab → add:
   BOT_TOKEN = (your token from BotFather)
   OWNER_CHAT_ID = 6408550462
4. Railway gives you a URL like: https://bitenowbot-production.up.railway.app

## Step 3 — Set webhook (paste in browser)
https://api.telegram.org/bot<YOUR_TOKEN>/setWebhook?url=https://YOUR-RAILWAY-URL.up.railway.app/webhook

Should say: {"ok":true}

## Step 4 — Test
Message @BiteNowBot on Telegram → /start
Go through the flow → order lands in YOUR Telegram 🔥
